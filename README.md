## How to run it ##
1. Clone repository: 
```shell
git clone https://github.com/1amL0st/rchat
```
2. Cd into repository: 
```shell
cd rchat
```
3. Build client:
```shell
cd client
yarn install
yarn build
cd ..
```
4. Run server:
```shell
cargo run
```
5. Open browser and go to 
```shell
localhost:8080
```
