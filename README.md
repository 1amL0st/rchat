## How to run it ##
1. Clone repository: 
```shell
git clone https://github.com/1amL0st/rchat
```
2. Cd into repository: 
```shell
cd rchat
```
3. Run server:
```shell
cargo run
```
4. Open browser and go to localhost:8080
